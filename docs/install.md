# Install

Install `ts-i18n` via your package manager of choice.

## Package Managers

::: code-group

```sh [bun]
bun add ts-i18n
```

```sh [npm]
npm i ts-i18n
```

```sh [pnpm]
pnpm add ts-i18n
```

```sh [yarn]
yarn add ts-i18n
```

:::

## Next steps

- Create your `locales` directory (Option A structure recommended)
- Optionally scaffold a config: `ts-i18n init`
- See the Usage page for examples
